# MetroMitra
No Maiden's Mini Project i.e. QR-AR Based Indoor Navigation System

PS: To implement a smart indoor navigation system for Metro commuters to provide an efficient wayfinding and optimal route to various access points

Mumbai Metro can be confusing for newcomers and tourists, especially when switching lines or finding exits.  

- **QR Codes:** Scan station-specific QR codes to trigger assistance.
- **AR Navigation:** Get augmented reality overlays that guide you through station interiors.
- **Interactive Metro Map:** Visualize lines, interchanges, and station layouts.
